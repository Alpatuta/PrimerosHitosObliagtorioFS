# Hito 2 - HTML y CSS estatico

Proyecto estatico para el segundo hito del Obligatorio 2.

## Pantallas

- `index.html#login`: pantalla estatica de ingreso.
- `index.html#registro`: pantalla estatica de registro.
- `index.html#dashboard`: dashboard estatico con alta de receta, listado, filtros, uso del plan y espacio reservado para grafico.

## Alcance

- No tiene JavaScript.
- No consume la API.
- No comparte logica con el Hito 1.
- No resuelve graficas, solo reserva el espacio visual.
- Usa HTML y CSS propios, sin componentes JS.
- Mantiene el formato de Single Page Application: un unico HTML con secciones internas.

## Como verlo

Abrir `index.html` en el navegador.

Tambien se puede levantar un servidor local desde esta carpeta:

```bash
python3 -m http.server 5501
```

Luego abrir:

```text
http://localhost:5501/index.html
```
